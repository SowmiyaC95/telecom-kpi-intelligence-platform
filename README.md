# 📡 Telecom Explainable AI Platform

🚀 End-to-end AI-based telecom analytics platform with:
- Anomaly Detection
- Explainable AI (SHAP RCA)
- Real-time Alerting
- Dashboard integration

---

## 🧠 Problem Statement

Telecom networks generate massive KPI data:
- Call Drop Rate (DCR)
- Call Setup Success Rate (CSSR)
- Latency / SINR / Throughput

👉 Detecting issues and identifying root causes manually is slow and inefficient.

✅ This project automates:
- Detection (AI)
- Diagnosis (Explainable AI)
- Alerting (Real-time)
